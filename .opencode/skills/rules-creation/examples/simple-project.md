# Simple Web Project

A personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## Project Structure

- `index.html` - Homepage
- `css/` - Stylesheets
- `js/` - JavaScript files
- `assets/` - Images and fonts

## Code Standards

- Use semantic HTML5 elements
- Mobile-first responsive design
- Vanilla JavaScript (no frameworks)
- CSS variables for theming in `:root`

## Development

- Open `index.html` directly in browser for testing
- Use Live Server extension for auto-reload
- Test on mobile viewport (Chrome DevTools)
